#### 백엔드
웹 사이트의 백엔드는 서버, 응용 프로그램 및 데이터베이스로 구성됩니다. \
백엔드 개발자는 이러한 구성요소들이 작동할 수 있게 하는 기술을 만들고 유지하는 일을 합니다. \
\
서버, 응용 프로그램, 데이터베이스가 서로 통신 할 수 있도록 만들기 위해 백엔드 개발자는 \
<b>PHP, Ruby, Python, Java, .Net과 같은 서버 측 언어</b>를 활용하여 응용 프로그램을 만듭니다. \
또한 데이터를 검색, 저장 또는 변경하고 이를 프론트엔드 코드로 사용자에게 \
다시 제공하기 위해서는 MySQL, Oracle 및 SQL Server를 사용합니다. 


#### 프론트엔드 
프론트엔드 개발자는 웹 사이트에서 사용자가 직접 경험하는 부분과 그 경험의 아키텍처를 담당합니다. \
이를 위해 프론트엔드 개발자는 HTML, CSS, JavaScript 활용에 능숙해야합니다. 
\
<b>백엔드에서 무슨 서버 측 언어를 사용하든, 프론트엔드에서 어떤 프레임워크를 사용하든, \
결과물은 무조건 HTML & CSS로 나오게 되기 때문에 이는 필수적으로 학습해야 합니다. </b>

#### 풀스텍
풀스택 개발자라는 호칭은 프론트엔드와 백엔드 모두에서 교차적으로 작업 할 수 있는 \
역할을 지칭하는 것에서 시작했습니다. 말 그대로 풀패키지(full package)를 제공하는 개발자라는 뜻입니다.
